🎯 模型选择

使用 Claude 3.5 Sonnet 模型，平衡了分析质量和响应速度。模型擅长理解代码上下文，能准确识别潜在问题。

🔍 上下文获取

通过 GitHub API 获取 PR 元数据、文件变更和代码差异。结合 PR 描述和 commit 信息，理解变更意图。

📊 误报控制

采用分级问题严重程度（高/中/低），区分安全、性能、逻辑等类型。提供详细的修复建议，减少误报。

🚀 扩展方向

支持更多 AI 模型、更多代码托管平台。可扩展自定义 Review 规则、团队协作、CI/CD 集成等功能。
